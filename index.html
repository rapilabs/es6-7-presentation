Comprehensions

  • Both array & generator comprehensions!
  • Format is for-if-statement, rather than statement-for-if
var someArray = [
  for (thing of things)
    if (thing.quack)
      thing.quack()
];

// as a generator:
var someArray = (
  for (thing of things)
    if (thing.quack)
      thing.quack()
);
